# Some Keybinds
|Keybind|Command|
|:-----:|:------:|
|super + m| Kill hyprland|
|super + t | Term|
|super + r | Launcher|
|super + f | Toggle fullscreen|
|super + v | Toggle floating mode|
|super + q | Close window|
|super + l | Lock screen|
|control + super + arrow| Change workspace|
